Stage 0
Important Concepts
The World Wide Web started early in the 1990s and has grown exponentially since then.
The Web consists of HTML documents, where hyperlinks connect the pages like a web.
HTML or hypertext markup language is the code that runs webpages like this one.
When a person goes to a web page like Google, their computer sends an HTTP request to a server.
The server finds the right HTML document, and sends it back to the person's computer to be displayed.

Tags
Tags are bits of HTML code that instruct the computer on what to do.
Many tags have an opening tag and a closing tag, but there are a expections to this rule, such as the
tag for a line break.
Tags tell the computer how an HTML document should look.

Inline or Block elements
Some HTML elements are known as inline because the formatting they create
appears inline with the text. Other elemetns are block elements because they make
an invisible box around the content that is placed inside the tags.

Computers need to be told
Computers are stupid in that they need very specific instructions to know what to do.
They literally take everything literally.

Stage 1

Tree like structure of HTML
HTML is structure a bit like a family tree. My parents had two kids, so there are two boxes underneath my parents box.
This forms the boxes of the webpage, which is the frame upon which all things on the web are hung.

Boxes, not Tubes
Everything on the web has a box that it fits into, and these boxes can be moved about on the page like pictures in a scrapbook.
There is an invisible grid that can be manipulated so make your HTML page look how you want it to look.

CSS
CSS stands for Cascading Style Sheets, which give coders a great way to control how their webpage looks.
By giving similar HTML elements the same class name, the CSS can tell the HTML what style it should use on that class.
